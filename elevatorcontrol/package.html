This page lists and describes the java files in this elevatorcontrol package.  This document is updated to reflect the current implementation.

Controller files

  • CarPositionControl.java - implements CarPositionIndicatorControl
  • CarButtonControl.java - implements CarButtonControl
  • Dispatcher.java - implements Dispatcher
  • DoorControl.java - implements DoorControl
  • DriveControl.java - implements DriveControl
  • HallButtonControl.java - implements HallButtonControl
  • LanternControl.java - implements LanternControl

Translator files

  • CarCallCanPayloadTranslator.java - implements an efficient boolean CAN translator for mCarCall
  • DesiredFloorCanPayloadTranslator.java - implements an efficient CAN translator for mDesiredFloor
  • DoorMotorCommandCanPayloadTranslator.java - implements an efficient CAN translator for mDoorMotor
  • DriveSpeedCanPaylodTranslator.java - implements an efficient CAN translator for mDriveSpeed
  • HallCallCanPayloadTranslator.java - implements an efficient boolean CAN translator for mHallCall

Runtime Monitors

  • Proj7RuntimeMonitor.java - implements required runtime monitor for Project 7.
  • RuntimeRequirementsMonitor.java - implements runtime monitor.

Utility files

  • Utility.java - implements AtFloor, DoorClosed, DoorOpened, Door Motor, Level, HallCall and CarCall utility functions.
  • MessageDictionary.java - Defines the messages and priorites for the CAN network.
